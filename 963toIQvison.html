Welcome to the convertion tool

This tool allows you to take the 963 Export of the schematics and cover them quickly and easily to a useful IQvision version

Intstuction

  • Follow the trend 963 export progress and make sure to import correctly into IQvision
  • Open the PX page in Text Editor view copy and paste every line that starts with '
  • Once it is done click the copy button at the bottom of the output page and paste it back into the text editor in IQvison
  • then change back to the PX editor and all being well great, if not contact support here

Exceptions

  • All time schedule (Z point) will not change and will still show as red flashing items on the page
  • As it stands in this version all the history will need to be manually changed. The point on the page is currently set up as all sensors have a historyOrd that will not work
  • All points display the label name and the point state if this needs to be changed this can be quickly done with find and replace in a text editor by finding '%displayName% ' and replacing it with nothing
  • Any issues at all submit them here
