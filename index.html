What is this?

This is a hub where cutting-edge software meets smart solutions! Whether you're an old-school BMS engineer or a freshly trained For N4 and IOT engineers or enthusiasts in smart building technology, we've crafted an experience that is tailored to you.
Our mission is to provide you with tools to take the time out of building station(supervisors) be it company templates to allow multiple engineers to stick to the same format every time or one-off system from subcontractors that want their own style easily with out having to start for scratch every time.
These tools aim to empower you to optimize your time to allow you to innovate instead of being stuck putting point by point on the page or by going through 100s of pages changing one setting.
Using these hand build .exe programs will allow you to change the PX page in Niagara (N4) on mass, which will save you time.
